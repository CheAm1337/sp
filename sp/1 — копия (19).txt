Вопросы: Что из себя представляет bash скрипт?  Определение: Текстовый файл с командами для оболочки Bash, используемый для автоматизации задач.

Как сделать так, чтобы скрипт задавал вопросы и на основе ответов формировал итоговую команду? С использованием read задаем вопросы, сохраняем ответы в переменные, формируем итоговую команду.
read -p "Имя пользователя: " username
read -p "Директория: " directory
command="sudo useradd $username -d $directory"
echo "Итоговая команда: $command"

Как передать аргументы из командной строки в скрипт?
Передача аргументов из командной строки:
Используем $1, $2, и так далее, чтобы получить аргументы, переданные из командной строки.
Пример:
В файле .sh
echo "Первый аргумент: $1"
echo "Второй аргумент: $2"

Вызов скрипта:
./myscript.sh arg1 arg2

Практическое задание:
Создайте функции просмотра информации о пользователе(1), создания пользователя(2), удаления пользователя (3). Используйте case и select для использования этих функций в виде меню.
# Функция просмотра информации о пользователе
nano название.sh

view_user_info() {
  read -p "Введите имя пользователя: " username
  id "$username"
}
create_user() {
  read -p "Введите имя нового пользователя: " new_username
  sudo useradd "$new_username"
  echo "Пользователь $new_username создан."
}
delete_user() {
  read -p "Введите имя пользователя для удаления: " del_username
  sudo userdel -r "$del_username"
  echo "Пользователь $del_username удален."
}
# Меню
PS3="Выберите действие (1 - Просмотр, 2 - Создание, 3 - Удаление, 4 - Выход): "
select option in "Просмотр" "Создание" "Удаление" "Выход"; do
  case "$REPLY" in
    1) view_user_info ;;
    2) create_user ;;
    3) delete_user ;;
    4) break ;;
    *) echo "Некорректный выбор." ;;
  esac
done
chmod +x название.sh
