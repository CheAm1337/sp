Вопросы: Для чего нужен загрузчик? Загрузчик необходим для старта операционной системы при запуске компьютера.
 Как изменить настройки загрузчика? Настройки загрузчика можно изменить редактированием файла конфигурации, например, для GRUB - /etc/default/grub.
 Для чего нужен loopback(lo) интерфейс (Работа с сетями)? Loopback (lo) - это виртуальный интерфейс, используемый для тестирования сетевых приложений на самом компьютере, а также для локальной разработки и отладки.

Практическое задание:
Отключите сервис sshd, перезапустите систему и посмотрите статус этого сервиса. Запустите сервис sshd и посмотрите его статус. Запретите сервису sshd запускаться.
Отключить sudo systemctl stop sshd
Перезапуск sudo reboot
Статус sudo systemctl status sshd
Запуск sudo systemctl start sshd
Статус sudo systemctl status sshd
Запрет sudo systemctl disable sshd
