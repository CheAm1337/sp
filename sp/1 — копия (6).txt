Вопросы: Какой синтаксис у команды cp, если нужно скопировать несколько файлов и директорий? Как можно использовать несколько ключей одновременно? Как можно использовать * и {} при работе с файлами?

1. Копирование нескольких файлов и директорий:   cp file1 file2 directory/ destination/
2. Использование нескольких ключей:   cp -r -i source destination
3.  - `cp *.txt destination/` (копирование всех файлов с расширением .txt).
 - `find . -name "*.txt" -exec cp {} destination/ \;` (копирование файлов с расширением .txt с использованием find).

Практическое задание: 
Создайте директорию /home/it. Создайте группу IT с идентификатором 10000. Создайте 3 пользователей - sysadmin, helpdesk и netadmin, домашние директории которых будут находиться в /home/it, а также они будут входить в группу IT и оболочкой у всех будет /bin/bash. Идентификаторы должны быть 10101, 10201, 10301.

sudo mkdir /home/it
sudo groupadd -g 10000 IT
sudo useradd -m -d /home/it/sysadmin -u 10101 -g IT -s /bin/bash sysadmin
sudo useradd -m -d /home/it/helpdesk -u 10201 -g IT -s /bin/bash helpdesk
sudo useradd -m -d /home/it/netadmin -u 10301 -g IT -s /bin/bash netadmin
-m: создать домашний каталог, если он не существует.
-d: указать домашний каталог пользователя.
-u: указать идентификатор пользователя.
-g: указать группу, к которой принадлежит пользователь.
-s: указать оболочку пользователя.
cat /etc/passwd – посмотреть пользователей
