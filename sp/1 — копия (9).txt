ЭКЗАМЕНАЦИОННЫЙ БИЛЕТ №10


Вопросы: Перечислите стандартные потоки. Как и куда можно направлять стандартные потоки? Как направить stdout и stderr в один файл и в разные файлы?

Стандартные потоки:
stdin (ввод): Получение данных.
stdout (вывод): Отправка обычных данных.
stderr (ошибки): Отправка сообщений об ошибках.

Направление потоков:
Стандартные потоки можно направлять в различные места с использованием операторов перенаправления в командной оболочке. Вот основные способы и направления:
Перенаправление ввода: command < input_file
Перенаправление вывода: command > output_file
Перенаправление ошибок: command 2> error_file
Добавление в конец файла (stdout): command >> output_file
Перенаправление stderr в stdout: command 2>&1
Перенаправление stdout и stderr в разные файлы: command > output_file 2> error_file
Направление в один файл:
stdout и stderr в один файл: command > combined_output_file 2>&1
Практическое задание:
Добавьте 2 диска по 1Гб к виртуалке. На первом диске создайте ТР MBR и 2 основных раздела по 70Мб, на втором ТР GPT и 3 раздела по 90Мб. Сохраните и покажите список разделов.
Добавьте на первом диске ещё 2 раздела по 100Мб, при этом не удаляя ранее созданные разделы.
 Удалите со второго диска второй раздел.
1. **Добавление 2 дисков по 1Гб к виртуальной машине:**
   - Виртуальная машина должна быть выключена.
   - Добавьте два новых диска с помощью инструментов вашего виртуализатора (например, VirtualBox, VMware).

2. **Создание разделов на первом диске:**
   - Включите виртуальную машину и убедитесь, что новые диски видны.
   - Используйте утилиту `fdisk` для создания разделов. Ниже представлен пример для первого диска:
     sudo fdisk /dev/sdb
     - Создайте таблицу разделов MBR (нажмите `o`).
     - Создайте два основных раздела по 70Мб каждый.
     - Сохраните изменения (нажмите `w`).
   - Используйте утилиту `gdisk` для создания разделов на втором диске:
     sudo gdisk /dev/sdc
     - Создайте таблицу разделов GPT (нажмите `o`).
     - Создайте три раздела по 90Мб каждый.
     - Сохраните изменения (нажмите `w`).
3. **Показать список разделов:**
   sudo fdisk -l
4.	**Добавление 2 разделов по 100Мб на первом диске:**
5.	   sudo fdisk /dev/sdb
   - Добавьте два новых основных раздела по 100Мб каждый.
5. **Удаление второго раздела со второго диска:**
   sudo gdisk /dev/sdc
   - Найдите номер второго раздела и удалите его.
   После завершения этих шагов, у вас должны быть созданы и удалены соответствующие разделы. Пожалуйста, убедитесь, что выполняете эти действия с осторожностью, чтобы избежать потери данных.
