Вопросы: Как узнать, есть ли определённый модуль в системе? 
lsmod | grep module_name

Для чего нужна директория /dev?
Назначение: Хранит файлы, представляющие устройства в системе.

Как загрузить и выгрузить модуль ядра?
sudo rmmod module_name

Практическое задание:
Сделайте скрипт, который будет спрашивать имя пользователя и выводить оболочку этого пользователя. Сделайте скрипт, который спросит имя пользователя и скажет, является ли пользователь администратором?
Nano чтобы создать файл.sh и выполнить это ./файл.sh
read -p "Введите имя пользователя: " username
shell=$(grep "^$username:" /etc/passwd | awk -F: '{print $7}')
if [ -n "$shell" ]; then
  echo "Оболочка пользователя $username: $shell"
else
  echo "Пользователь $username не найден."
fi
read -p "Введите имя пользователя: " username
if groups "$username" | grep -q '\bsudo\b'; then
  echo "Пользователь $username является администратором."
else
  echo "Пользователь $username не является администратором."
