Вопросы: Чем скрытые файлы отличаются от обычных? Для чего нужен ключ -v (verbose)? Зачем нужен ключ -i у команды cp?
1. Скрытые файлы начинаются с точки в имени (например, `.bashrc`). Они не отображаются по умолчанию при просмотре содержимого каталога.
2. Ключ -v (verbose) в командах (например, `cp -v`) выводит подробную информацию о выполняемых операциях.
3. Ключ -i в команде `cp` предотвращает случайное перезаписывание файлов, запрашивая подтверждение перед заменой.

Практическое задание: 
С помощью переменных докажите, что login shell и non-login shell отличаются.
echo "Это login shell: $0"

Сделайте пользователя user2 администратором и докажите, что это работает
sudo useradd -m user2
sudo usermod -aG sudo user2
groups user2 тут смотрим что есть sudo
Разрешите пользователю user1 запускать команду touch от пользователя user2 и докажите, что это работает. 
# Создаем файл, который будет доступен только для записи пользователю user2.
sudo touch /tmp/example_file
sudo chown user2 /tmp/example_file
sudo chmod 700 /tmp/example_file

# Добавляем пользователя user1 в группу, которой принадлежит файл.
sudo usermod -aG user2 $(whoami)
